### Getting Started

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd HA-SY-DAM-000001
    ```

2.  **Configure Git Credentials:**
    To avoid entering your username and password for every push, you can configure Git to store your credentials.
    ```bash
    git config --global credential.helper store
    ```
    The first time you push, you will be asked for your credentials, and they will be saved for future use.

3.  **Start Contributing:**
    -   Edit the configuration files in the `config/` directory.
    -   Use `git add`, `git commit`, and `git push` to track your changes.
